This project demonstrates how to deploy a static web server using Nginx inside a Docker container.

It also includes real-time JavaScript features such as a live clock, countdown timer, and dark mode toggle.

All content is served securely and efficiently using Docker volumes.

Security Notice

This demo follows best practices like the use of a security.txt file, minimal privileges, and read-only mounts.
